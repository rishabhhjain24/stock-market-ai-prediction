# AI Trading Forecast Dashboard - Complete Trading Guide

## 📊 How to Read and Trade Using the Dashboard

### 1. **TOP STATUS INDICATORS**

#### Market Status
- 🔴 **MARKET CLOSED** = No live trading (analysis only)
- ✅ **MARKET OPEN** = Ready for live trades (9:15 AM - 3:30 PM IST for NSE)

**Action**: Only execute trades during market hours shown.

---

### 3. **UNIFIED AI FORECAST** (Main Section)

This is the core trading signal. Look at:

#### **Current Price**
Shows the current trading price of the stock

#### **Recommendation** (CRITICAL)
```
Options: BUY | SELL | HOLD | STRONG BUY | STRONG SELL
```

**How to Interpret**:
- **STRONG BUY** (90%+ confidence) → Enter with full position
- **BUY** (70-89% confidence) → Enter with medium position
- **HOLD** (40-70% confidence) → Wait or take small position
- **SELL** (70-89% confidence) → Exit or short (if allowed)
- **STRONG SELL** (90%+ confidence) → Exit immediately or go short

#### **Confidence %**
Shows how sure the AI is about the recommendation.
- **70%+** = Strong signal, take trade
- **50-70%** = Medium signal, be careful
- **<50%** = Weak signal, skip trade

#### **Signal Strength**
Shows the intensity of the signal (0-100%).
- **70%+** = Strong move expected
- **50-70%** = Medium move expected
- **<50%** = Small move or uncertain

#### **Risk/Reward Ratio**
Example: `0.00:1` means risk vs potential profit
- **1:2** = Risk ₹100 to make ₹200 (GOOD)
- **1:3** = Risk ₹100 to make ₹300 (EXCELLENT)
- **2:1** = Risk ₹200 to make ₹100 (BAD - AVOID)

### 4. **PRICE TARGETS & ENTRY/EXIT POINTS**

```
Entry Point: ₹264.06 (Where to buy)
1h Target: ₹264.06 (Target in 1 hour)
4h Target: ₹264.06 (Target in 4 hours)
1d Target: ₹264.06 (Target in 1 day)
```

**How to Use**:
1. **Entry**: Buy at or near the entry price
2. **Take Profit**: Sell at targets (take partial profits at each target)
3. **Stop Loss**: Use the stop loss in Risk Management section

**Example Trade**:
```
Entry: Buy at ₹264.06
Stop Loss: ₹264.06 (Risk Management section)
Target 1: Sell 30% at ₹264.06 (1h target)
Target 2: Sell 40% at ₹264.06 (4h target)
Target 3: Sell 30% at ₹264.06 (1d target)
```

### 5. **RISK MANAGEMENT** (MANDATORY)

#### Stop Loss
The price where you MUST exit if wrong.
- Set as a hard limit in your trading app
- Never ignore this

#### Portfolio Risk %
Shows how much of your portfolio is at risk.
- **1-2%** = Conservative (GOOD)
- **2-5%** = Moderate
- **>5%** = Aggressive (RISKY)

**How to Use**:
```
If Portfolio Risk = 1.0%
Your Portfolio = ₹1,00,000
Max Risk = ₹1,000 per trade

Calculate Position Size:
Position Size = (₹1,000) / (Entry - Stop Loss)
```

## 🎯 STEP-BY-STEP TRADING WORKFLOW

### **BEFORE YOU TRADE**

1. **Check Market Status**
   - Is market open? ✅
   - Are there any market disruptions? ✅

2. **Check Global Context**
   - Global markets bullish or bearish?
   - Is there misalignment? ✅

3. **Read the Recommendation**
   - Is it BUY/SELL/HOLD? ✅
   - Is confidence >70%? ✅
   - Is risk/reward ratio >1:1? ✅

### **WHEN YOU SEE A BUY SIGNAL**

```
STEP 1: Verify Setup
✓ Recommendation = BUY or STRONG BUY
✓ Confidence > 70%
✓ Technical Score + Price Score > 0.20

STEP 2: Check Risk Management
✓ Calculate position size = (Capital × Risk %) / (Entry - Stop Loss)
✓ Ensure risk ≤ 2% of portfolio
✓ Set stop loss order in your broker

STEP 3: Enter Trade
✓ Buy at or near Entry Point
✓ Set stop loss at Stop Loss price (MANDATORY)
✓ Set take profit at targets

STEP 4: Monitor
✓ Watch 1h target - take 30% profit
✓ Watch 4h target - take 40% profit
✓ Let 30% run to 1d target
✓ Or use trailing stop loss

STEP 5: Exit
✓ Hit target = SELL (take profit)
✓ Hit stop loss = EXIT (cut loss)
✓ Reached max time = CLOSE (time decay)
```

### **WHEN YOU SEE A SELL SIGNAL**

```
STEP 1: Verify Setup
✓ Recommendation = SELL or STRONG SELL
✓ Confidence > 70%

STEP 2: Exit or Short
Option A (Conservative): Exit existing long positions
Option B (Aggressive): Short if allowed by broker

STEP 3: Set Stop Loss
✓ For short: Stop loss ABOVE entry
✓ Use same risk management rules

STEP 4: Take Profits
✓ Same as BUY (at targets)
```

### **WHEN YOU SEE HOLD**

```
STEP 1: Analyze Components
✓ Which signals are strong?
✓ Which are weak?

STEP 2: Wait for Confirmation
✓ Hold existing positions
✓ Don't enter new trades
✓ Watch for signal to change

STEP 3: Watch the Confidence
✓ If confidence increasing → Signal turning
✓ If confidence decreasing → Signal weakening
```

## ⚠️ CRITICAL RULES

### **ALWAYS DO THIS**
1. ✅ Set stop loss BEFORE entering trade
2. ✅ Risk only 1-2% per trade
3. ✅ Take profits at targets
4. ✅ Check confidence > 70% for strong signals
5. ✅ Verify risk/reward > 1:1

### **NEVER DO THIS**
1. ❌ Trade without stop loss
2. ❌ Risk >5% per trade
3. ❌ Ignore the stop loss
4. ❌ Trade when confidence <50%
5. ❌ Average down on losing trades
6. ❌ Trade when market is closed
