## 🐛 IF SOMETHING BREAKS

### "Deploy won't start"
- Missing `requirements.txt`? 
- Solution: `pip freeze > requirements.txt && git push`

### "Crashes when deployed"
- Missing dependency
- Solution: Check logs, add to requirements.txt

### "API keys don't work"
- Not added to Secrets
- Solution: Add to Settings → Secrets in deployment platform

### "Can't access from phone"
- Wrong URL
- Solution: Copy URL from deployment platform

**All troubleshooting in:** `DEPLOY_COMPLETE_GUIDE.md`
