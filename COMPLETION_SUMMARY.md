## ⚠️ CRITICAL SAFETY RULES

### **🔴 NEVER BREAK THESE:**

1. **ALWAYS SET STOP LOSS BEFORE ENTRY**
   - Non-negotiable
   - Use level from dashboard
   - Never skip this

2. **NEVER MOVE STOP LOSS DOWN**
   - Locks in losses
   - Breaks risk management
   - Path to ruin

3. **NEVER TRADE OUTSIDE MARKET HOURS**
   - Check 🟢 MARKET OPEN on dashboard
   - Only: 9:15 AM - 3:30 PM IST
   - Wait if market closed

4. **NEVER RISK MORE THAN 2%**
   - Per trade limit
   - Formula: Position = (2% Account) / Risk
   - Respect the math

5. **NEVER TRADE WITHOUT JOURNAL**
   - Record every trade
   - Entry, exit, P&L
   - Weekly review essential

## 📱 QUICK REFERENCE

### **Dashboard Symbols:**
- 🟢 Green = Bullish/Open/Good
- 🔴 Red = Bearish/Closed/Warning
- ✅ Check = Complete/Working
- ❌ X = Skip/Warning
- ⚡ Lightning = Important
- 📊 Chart = Data/Metrics

### **AI Component Guide:**
- +0.5 to +1.0 = Strong bullish
- +0.2 to +0.5 = Mild bullish
- -0.2 to +0.2 = Neutral
- -0.5 to -0.2 = Mild bearish
- -1.0 to -0.5 = Strong bearish

### **Confidence Levels:**
- 75%+ = Strong signal (take it)
- 50-75% = Moderate signal (OK)
- 30-50% = Weak signal (be careful)
- <30% = Very weak (skip it)
